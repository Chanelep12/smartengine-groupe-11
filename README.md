## Stack technique
- Gemini CLI — orchestration des agents IA
- Python / pandas — manipulation des données
- scikit-learn — modélisation prédictive
- Streamlit — dashboard interactif
- n8n — automatisation des alertes
- GitHub — gestion du code source

## Structure du dépôt
```
smartengine-groupe-11/
├── .gemini/agents/        # Agents Gemini CLI
├── data/
│   ├── raw/               # CSV RavenStack (non commités)
│   └── processed/         # Données transformées (Sprint 2)
├── docs/                  # Livrables et documentation
├── outputs/               # Rapports générés par les agents
├── src/                   # Scripts Python
├── GEMINI.md              # Contexte permanent de l'agent
└── README.md
```
